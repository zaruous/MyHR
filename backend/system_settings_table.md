# `system_settings` 테이블 정보

이 테이블은 시스템의 다양한 설정값들을 키(key)와 값(value)의 형태로 저장합니다. 특히 급여 계산식에 사용되는 보너스, 공제액 등의 설정값을 관리하는 데 사용됩니다.

## 테이블 구조

| 컬럼명        | 타입         | 제약조건        | 설명                                   |
| :------------ | :----------- | :-------------- | :------------------------------------- |
| `setting_key` | VARCHAR(255) | PRIMARY KEY     | 설정의 고유 키 (예: `payroll_bonus`)   |
| `setting_value` | VARCHAR(255) | NOT NULL        | 설정 값 (예: `100000`)                 |
| `description` | TEXT         |                 | 설정에 대한 상세 설명                  |

## 데이터 예시

`system_settings` 테이블에 저장될 수 있는 데이터의 예시는 다음과 같습니다:

| setting\_key         | setting\_value | description                  |
| :------------------- | :------------- | :--------------------------- |
| `payroll_bonus`      | `100000`       | 전직원 고정 보너스 금액      |
| `payroll_deductions` | `300000`       | 전직원 고정 공제액           |

## 사용 목적

*   급여 계산식에 사용되는 변동 가능한 상수(보너스, 공제액 등)를 관리합니다.
*   관리자 페이지를 통해 해당 설정값을 동적으로 조회하고 수정할 수 있도록 합니다.
*   하드코딩된 값을 제거하여 시스템의 유연성을 높입니다.
